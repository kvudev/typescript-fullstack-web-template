# Setup database and ORM

## Database

- PostgreSQL

## ORM

- Prisma

## Database Structure
- `videos` table:
  - `id`: primary key
  - `video_id`: YouTube video ID
  - `title`: video title
  - `description`: video description
  - `published_at`: video publish date
  - `generated_content`: content generated by Gemini based on the video
  - `created_at`: record creation timestamp
  - `updated_at`: record update timestamp

## Docker
- Add PostgreSQL service to `docker-compose.yml` with appropriate environment variables for username, password, and database name.
- Create a Dockerfile for the database if needed (or use official PostgreSQL image).

## Prisma Setup
- Initialize Prisma in the backend directory: `npx prisma init`
- Configure `schema.prisma` to connect to the PostgreSQL database and define the `videos` model.
- Generate Prisma client: `npx prisma generate`
