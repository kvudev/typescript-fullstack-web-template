Feature name: News Details Popup

## Feature Description:
Implement a feature that allows users to click on a news item in the feed and view more details about it in a popup modal. The popup will display the full summary of the news article, along with any additional information such as the source, publication date, and a link to the original article. The popup will be designed to be visually appealing and easy to read, with a focus on providing a seamless user experience. This feature will enhance the usability of the news feed and allow users to quickly access more information about the news items that interest them.
